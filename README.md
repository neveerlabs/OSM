## 📦 Tentang Repositori

Repositori ini berisi **perjalanan pengembangan** *MySQL Operations Tool (OSM)* – sebuah alat berbasis Node.js untuk mengelola database MySQL langsung dari terminal.  
Setiap folder menyimpan versi yang berbeda, dari yang paling awal hingga yang terbaru.

| Versi | Deskripsi | Teknologi |
|-------|-----------|-----------|
| **v1** | Versi awal dengan readline – fungsional, minimalis. | `mysql2`, `readline` |
| **v2** | Versi interaktif dengan inquirer – lebih user‑friendly dan aman. | `mysql2`, `inquirer` |
| *(dan seterusnya)* | ... | ... |

> *Setiap versi bisa dijalankan secara independen. Cocok untuk belajar, eksperimen, atau keperluan produksi ringan.*

---

## Cara Menjelajah

1. Clone repositori ini:  
   ```bash
   git clone https://github.com/nflora-ux/Operating-System-MySQL.git
   ```
2. Masuk ke dalam folder versi yang diinginkan:
   ```bash
   cd Operating-System-MySQL/Operating-System-MySQL-v2   # contoh
   ```
3. Install dependensi:
   ```bash
   npm install
   ```
4. Jalankan aplikasi:
   ```bash
   node OSM.js
   ```

## Mengapa Repositori Ini Unik?
- Evolusi kode – lihat bagaimana sebuah tool berkembang dari sederhana menjadi powerful.
- Belajar dari praktik nyata – cocok untuk developer yang ingin memahami perbedaan pendekatan.
